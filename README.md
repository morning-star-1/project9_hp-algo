# High-Performance Pathfinding (A* and Weighted A*)

A Python implementation of A* and Weighted A* with deterministic benchmarks on grid maps to compare speed and solution quality.

## Quickstart
### Prerequisites
- Python 3.11+

### Run benchmarks
```bash
python benchmark.py
```

## Files
- `pathfinding.py` core algorithms
- `benchmark.py` benchmark runner

## Tests
```bash
python -m unittest discover -s tests
```
