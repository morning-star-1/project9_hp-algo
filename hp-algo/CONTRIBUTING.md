# Contributing

Thanks for your interest in contributing.

## Getting Started
- Follow the Quickstart in README.md.
- Use the project-specific scripts and commands in the README.

## Development Guidelines
- Keep changes focused and small.
- Add or update tests when behavior changes.
- Run formatting and linting before opening a PR.

## Formatting and Linting
- Node projects: `npm run lint`
- Python projects: `python -m ruff check .` and `python -m ruff format .`

## Pull Requests
- Describe the problem and the solution.
- Include screenshots or logs when relevant.
- Note any breaking changes.
